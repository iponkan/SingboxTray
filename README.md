## Singbox 托盘工具

本项目是一个专为 Windows 用户设计的 [sing-box](https://github.com/SagerNet/sing-box) 托盘管理工具，支持配置自动下载与更新。

### 📂 目录结构

```text
SingboxTray/
│
├── SingboxTray.bat       <-- 初始化/管理脚本 (通过此管理快捷方式)
├── url.conf              <-- [配置文件] (必须位于根目录) 存放 windows.json 的下载地址
│
└── core/                 <-- [核心文件夹]
    ├── sing-box.exe      <-- sing-box 程序
    ├── windows.json      <-- 实际使用的配置文件 (可自动下载)
    └── tray.ps1          <-- 托盘管理逻辑
```

### ✨ 功能特性

- **自动配置更新**：每次启动程序（如从桌面快捷方式打开），将自动从 `url.conf` 中的地址拉取更新 `windows.json` 配置文件。
- **一键重新启动**：托盘右键菜单提供“重新启动”选项，点击后会**先拉取最新远程配置**，再重启 sing-box。
- **系统托盘图标**：提供托盘图标，支持快捷管理，隐藏黑色命令行窗口。
- **UI 面板访问**：快速访问 Web UI (`http://127.0.0.1:9090/ui/`)。
- **快捷管理**：支持一键创建桌面快捷方式及设置开机自启。

### 🚀 使用方法

1.  **配置下载地址**：在项目**根目录**创建 `url.conf` 文件（**务必确保在根目录，而非 core 文件夹内**），填入你的 `windows.json` 直链下载地址。
2.  **启动程序**：双击运行根目录的 `SingboxTray.bat`，选择 `[1] 启动 Singbox (托盘模式)`。
3.  **自动更新配置**：程序每次启动时，都会根据根目录下的 `url.conf` 自动拉取最新的远程配置文件（下载失败时使用本地旧配置）。
4.  **快捷方式**：使用 `SingboxTray.bat` 的选项创建桌面快捷方式，方便后续直接启动。
5.  **更新配置**：如需同步远程配置的更改，只需右键点击托盘图标并选择 **“重新启动”**。

### ⚠️ 兼容性说明

- 无需开启 Windows “使用 Unicode UTF-8 提供全球语言支持”。PowerShell 脚本应以 **UTF-8 with BOM** 保存，避免 Windows PowerShell 5.1 在默认系统编码下把中文字符串解析成乱码。
- `tray.ps1` 会自动检测管理员权限；如果当前不是管理员，会触发 UAC 重新启动自身。用户取消 UAC 时会看到明确提示。
